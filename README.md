# 💰 Controle de Gastos

Aplicação web simples para controle de despesas pessoais, desenvolvida com foco em prática de JavaScript, manipulação do DOM e armazenamento local.

---

## 🚀 Acesse o projeto

https://lucasanddevrj.github.io/controle-gastos/

---

## 📌 Funcionalidades

* ➕ Adicionar gastos
* ❌ Remover gastos
* 💾 Persistência de dados com LocalStorage
* 📊 Visualização em gráfico (pizza)
* 💰 Cálculo automático do total

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (puro)
* Chart.js

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

* Praticar JavaScript na prática
* Trabalhar com manipulação de DOM
* Entender persistência de dados no navegador
* Criar projetos reais para portfólio
